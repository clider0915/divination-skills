# 🔮 算命能力合集 (Divination Skills)

## 简介

本技能包整合中华传统术数和西方性格分析体系，让 OpenClaw、Claude、GPT 等 AI 助手能够进行专业的算命起卦和性格分析服务。

### 中华传统术数

| 术数 | 适用场景 | 优势 |
|------|----------|------|
| **六爻问卦** | 一事一断、简单预测、近期决策 | 快速直观、门槛低 |
| **奇门遁甲** | 择时决策、方位选择、趋吉避凶 | 时空结合、行动指南 |
| **紫微斗数** | 命盘分析、人生规划、长期运势 | 全面系统、结构清晰 |

### 西方性格分析 [NEW]

| 方法 | 适用场景 | 优势 |
|------|----------|------|
| **占星学** | 性格分析、情感模式、事业规划 | 精准细致、时空结合 |
| **九型人格** | 动机分析、个人成长、关系模式 | 深度洞察、成长导向 |
| **MBTI** | 认知功能、职业匹配、团队建设 | 实用性强、应用广泛 |

### 🆕 变体方法

**时间起卦 + 六爻断卦**：用梅花易数的时间起卦法快速得卦，套用六爻纳甲体系深度断卦。

| 对比 | 传统六爻 | 时间起卦+六爻 |
|------|---------|--------------|
| **起卦方式** | 铜钱6次 | 时间计算 |
| **动爻数量** | 0-6个 | 固定1个 |
| **断卦体系** | 纳甲六亲 | 纳甲六亲 |
| **适用场景** | 深度分析 | 快速判断 |

**特点**：起卦快 + 断卦深 = 实用性强

## 快速开始

### 安装到 OpenClaw

```bash
# 方法1: 直接克隆到技能目录
cd ~/.openclaw/workspace/skills
git clone https://github.com/mingkunyuan/divination-skills.git divination

# 方法2: 下载压缩包
curl -L https://github.com/mingkunyuan/divination-skills/archive/main.tar.gz | tar xz
mv divination-skills-main ~/.openclaw/workspace/skills/divination
```

### 使用方式

重启 OpenClaw 或刷新技能后,直接对话:

```
用户:帮我起个卦看看这件事能不能成
用户:用奇门看看今天适合去哪个方向
用户:帮我排个紫微命盘,我是1990年5月15日早上8点出生的男生
```

## 示例对话

### 六爻问卦
```
用户：帮我看看下周面试能不能过
助手：好的，这是一个近期决策问题，适合用六爻来预测。
     请确认一下：面试是下周几？你想要“直接结论”还是“详细分析”？
```

### 六爻 - 时间起卦（快速模式）
```
用户：用时间起卦看看这件事能不能成
助手：好的，用当前时间起卦...
     
     起卦时间：2026年4月6日 10:55（巳时）
     本卦：地水师
     变卦：雷水解（四爻动）
     
     正在装卦分析...
```

**时间起卦特点**：
- ⚡ 起卦快速（无需铜钱）
- 🎯 动爻单一（判断清晰）
- 📊 保留六爻深度分析

### 占星学 [NEW]
```
用户：帮我看看星盘，我是什么性格
助手：好的，需要一些信息：
     1. 出生日期（公历还是农历？）
     2. 具体出生时间（最好精确到分钟）
     3. 出生地点
```

### 九型人格 [NEW]
```
用户：我是什么九型类型？
助手：让我问几个问题帮你识别：
     1. 你最大的恐惧是什么？
        - 犯错 → 可能是1型
        - 不被爱 → 可能是2型
        - 失败 → 可能是3型
        ...
```

### MBTI [NEW]
```
用户：我是什么 MBTI 类型？
助手：让我问几个问题：
     1. 聚会后你是充满能量还是精疲力尽？
        - 充满能量 → E（外向）
        - 精疲力尽 → I（内向）
     2. 你更关注细节还是整体？
        ...

### 奇门遁甲
```
用户:用奇门帮我看看今天去哪个方向谈合作比较好
助手:好的,奇门可以帮你选择方位。
     请告诉我:1)你在哪个城市?2)大概几点出发?
```

### 紫微斗数
```
用户:帮我排个紫微命盘
助手:好的,排紫微命盘需要一些信息:
     1. 出生日期是公历还是农历?
     2. 具体年月日是?
     3. 出生时间尽量精确到时辰
     4. 性别
```
