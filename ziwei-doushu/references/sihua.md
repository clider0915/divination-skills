# 紫微斗数四化详解

## 基础概念

四化是紫微斗数中最重要的动态机制，代表星曜受到天干的影响而产生质变。

### 四化含义

| 四化 | 符号 | 核心含义 | 类比 |
|------|------|---------|------|
| 化禄 | 禄 | 财禄、顺利、增加、得到 | 收入增加、机会来临 |
| 化权 | 权 | 权力、掌控、努力、竞争 | 升职加权、主动争取 |
| 化科 | 科 | 名声、贵人、科甲、平顺 | 声名远播、逢凶化吉 |
| 化忌 | 忌 | 纠缠、执着、不顺、损失 | 困扰、挫折、放不下 |

### 四化的层次
- **生年四化（本命四化）**：影响一生，为先天格局的一部分
- **大限四化**：影响十年，以大限宫干飞四化
- **流年四化**：影响一年，以流年天干飞四化

### 重要原则
- 化禄、化权、化科为三吉化，化忌为凶化
- 化忌不一定全凶，要看忌入何宫、忌何星
- 同一颗星同时化禄又化忌（不同层次），代表该领域有得有失

## 四化入宫含义

### 化禄入各宫

| 宫位 | 含义 |
|------|------|
| 命宫 | 一生多福禄，个性乐观，人缘好 |
| 兄弟 | 与兄弟姐妹感情好，手足有助力 |
| 夫妻 | 感情顺利，配偶带财或有助力 |
| 子女 | 子女有出息，桃花运好，投资运佳 |
| 财帛 | 财运亨通，赚钱容易，正财丰厚 |
| 疾厄 | 身体底子好，有口福享受 |
| 迁移 | 出外有贵人，外出发展好 |
| 交友 | 朋友多且有助，下属得力 |
| 官禄 | 事业顺利，工作有成就感 |
| 田宅 | 不动产运好，居家环境佳 |
| 福德 | 精神愉快，兴趣广泛，享福 |
| 父母 | 父母有助力，长辈缘好，学业好 |

### 化权入各宫

| 宫位 | 含义 |
|------|------|
| 命宫 | 个性强势，有主见，领导力强 |
| 夫妻 | 配偶能干但强势，婚姻中有权力角力 |
| 财帛 | 通过努力赚钱，理财能力强 |
| 官禄 | 事业上有实权，掌管实务 |

### 化科入各宫

| 宫位 | 含义 |
|------|------|
| 命宫 | 为人斯文有气质，有贵人运 |
| 官禄 | 事业有声望，名声好 |
| 父母 | 父母有文化，自己学业运好 |

### 化忌入各宫

| 宫位 | 含义 |
|------|------|
| 命宫 | 人生多烦恼，容易钻牛角尖，但也代表执着认真 |
| 夫妻 | 感情波折多，婚姻有考验，但也代表在意 |
| 财帛 | 财运有阻碍，容易破财或财来财去 |
| 官禄 | 事业多波折，工作压力大，但也代表用心 |

### 化忌的深层理解
化忌不全是凶——它代表"在意""执着""课题"：
- 化忌入夫妻：很在意感情，因为在意所以纠结
- 化忌入官禄：非常重视事业，因为重视所以压力大
- 化忌可以理解为"这辈子的功课"，是需要去修炼和成长的领域

## 大限四化与流年四化

### 大限四化
以大限所在宫位的天干飞四化，叠加在本命盘上分析。

**叠加效应**：
| 组合 | 含义 |
|------|------|
| 本命化禄 + 大限化禄 | 双禄临门，该领域大发 |
| 本命化禄 + 大限化忌 | 禄忌交战，先好后坏或有得有失 |
| 本命化忌 + 大限化忌 | 双忌叠加，该领域压力极大 |
| 本命化忌 + 大限化禄 | 忌处逢禄，困境中有转机 |

### 流年四化
以流年天干飞四化，叠加在大限和本命盘上。

流年命宫的确定：流年地支所在的宫位即为流年命宫。

### 三层四化综合分析步骤
1. 先看本命四化定先天格局
2. 叠加大限四化看十年运势
3. 再叠加流年四化看当年状况
4. 重点关注三层四化是否集中在同一宫位——集中之处必是该年焦点
