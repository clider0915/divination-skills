# 紫微斗数星曜详解

## 主星总览

十四主星分为北斗星系和南斗星系：

### 北斗星系（紫微星系）
| 星名 | 核心关键词 | 类比角色 |
|------|-----------|---------|
| 紫微 | 帝王、尊贵、统御、好面子 | 皇帝 |
| 天机 | 智慧、机变、谋略、善变 | 军师 |
| 太阳 | 光明、博爱、贵气、付出 | 太阳/慈父 |
| 武曲 | 财星、刚毅、果断、孤克 | 财务总监 |
| 天同 | 温和、享福、懒散、晚发 | 福星/小孩 |
| 廉贞 | 复杂、桃花、囚星、政治 | 次桃花/政客 |

### 南斗星系（天府星系）
| 星名 | 核心关键词 | 类比角色 |
|------|-----------|---------|
| 天府 | 稳重、库府、保守、衣食 | 财政大臣/国库 |
| 太阴 | 清秀、田宅、柔美、母亲 | 月亮/慈母 |
| 贪狼 | 欲望、桃花、才艺、多变 | 交际花/艺术家 |
| 巨门 | 口才、是非、暗星、分析 | 律师/评论家 |
| 天相 | 辅佐、印星、衣食、随和 | 宰相/秘书长 |
| 天梁 | 荫星、化解、老成、清高 | 法官/长者 |
| 七杀 | 将星、冲劲、孤独、变动 | 将军/冒险家 |
| 破军 | 破坏、开创、消耗、变革 | 先锋官/革命者 |

## 主星详解

### 紫微星
**核心特质**：尊贵、自尊心强、领导力、好面子
**优点**：气度恢宏、有领袖气质、慷慨大方
**缺点**：自视甚高、主观意识强
**在各宫含义**：
- 命宫：天生具有领导气质，自尊心极强
- 官禄：事业格局大，适合管理层或创业
- 财帛：财运好但花费也大，讲究排场

### 天机星
**核心特质**：聪明、善变、谋略、浮动
**优点**：思维敏捷、分析力强、善于规划
**缺点**：想太多做太少、犹豫不决
**在各宫含义**：
- 命宫：聪明伶俐但容易想多做少
- 官禄：适合企划、策略、顾问类工作

### 太阳星
**核心特质**：光明正大、博爱、付出型、贵气
**优点**：热情大方、乐于助人、有正义感
**缺点**：不聚财、付出过多、劳碌
**特殊**：太阳在寅卯辰巳午未宫为"日出到日中"，力量递增；在申酉戌亥子丑宫力量递减

### 武曲星
**核心特质**：财星第一、刚毅果断、实干、孤克
**优点**：执行力极强、财运好、做事干脆
**缺点**：性格孤僻、缺乏人情味
**在各宫含义**：
- 命宫：刚直务实，适合金融财经
- 财帛：正财运极佳，善于理财

### 天同星
**核心特质**：福星、温和、享受、懒散、晚发
**优点**：性格温和、与人为善、有福气
**缺点**：缺乏上进心、依赖性强
**特殊**：天同化忌反而可以激发斗志，属于"逆境成才"型

### 廉贞星
**核心特质**：复杂多面、政治手腕、次桃花、囚星
**优点**：精明能干、应变能力强、善于交际
**缺点**：情绪波动大、固执己见

### 天府星
**核心特质**：稳重可靠、国库、保守、衣食无忧
**优点**：稳健踏实、有安全感、善于守成
**缺点**：过于保守、缺乏开创精神
**关键**：天府被称为"令星"，其力量受对宫和夹宫影响大

### 太阴星
**核心特质**：柔美、田宅、母亲/妻子、内敛
**优点**：细腻温柔、有审美感、田宅运好
**缺点**：多愁善感、优柔寡断
**特殊**：在申酉戌亥子丑宫为"月升到月满"，力量递增

### 贪狼星
**核心特质**：欲望、桃花、才艺、社交、多才多艺
**优点**：才华横溢、社交能力极强
**缺点**：欲望太多、沉迷享乐
**特殊**：贪狼遇火星或铃星（火贪格/铃贪格）反而是大吉格局，主突发横财

### 巨门星
**核心特质**：口才、是非、暗星、分析能力
**优点**：口才极佳、分析能力强
**缺点**：是非多、猜疑心重
**特殊**：巨门化禄为"石中隐玉"，化忌则是非加倍

### 天相星
**核心特质**：辅佐、印星、衣食、随和
**优点**：协调能力强、注重礼仪
**缺点**：缺乏主见、依赖性强
**关键**：天相受夹宫影响极大——被吉星夹（财荫夹印）则大吉

### 天梁星
**核心特质**：荫星、化解灾厄、清高、老成持重
**优点**：逢凶化吉、正义感强
**缺点**：过于清高、好为人师
**在各宫含义**：
- 命宫：有化解灾厄的能力，适合公益
- 官禄：适合医疗、法律、宗教

### 七杀星
**核心特质**：将星、冲劲十足、孤独、变动
**优点**：魄力十足、勇于开创
**缺点**：性格孤傲、人际关系差
**在各宫含义**：
- 命宫：人生起伏大，大起大落
- 官禄：适合军警、开创性事业

### 破军星
**核心特质**：破坏与重建、先锋、消耗、变革
**优点**：开创力极强、勇于变革
**缺点**：破坏性强、不够稳定
**提示**：破军不只代表"破"，也代表重组、转型、换赛道

## 六吉星

| 星名 | 核心特质 | 在命宫含义 |
|------|---------|-----------|
| 文昌 | 正统学问、科甲功名 | 聪慧好学、学历高 |
| 文曲 | 才艺、口才 | 多才多艺、有文艺天赋 |
| 左辅 | 助力、贵人 | 人缘好、有贵人帮 |
| 右弼 | 助力、贵人 | 同上，更柔和 |
| 天魁 | 阳贵人 | 遇难有人帮，考试运好 |
| 天钺 | 阴贵人 | 暗中有贵人相助 |

## 六煞星

| 星名 | 核心特质 | 在命宫含义 |
|------|---------|-----------|
| 擎羊 | 刑伤、是非、刚烈 | 个性刚强、争斗 |
| 陀罗 | 拖延、纠缠 | 做事拖拉、暗中消耗 |
| 火星 | 急躁、暴烈 | 性急、脾气暴 |
| 铃星 | 阴火、暗燃 | 心火暗烧、内心焦虑 |
| 地空 | 空虚、落空 | 想法独特但不切实际 |
| 地劫 | 劫夺、损失 | 财物损失，但有灵感 |

煞星不全是凶——在特定组合中反而成为催化剂。
